# Claude Code Rules — LeoPortfolio

## Syntax checks (MANDATORY)
- Run `node --check app.js` before AND after every set of JS edits
- Never use curly/typographic quotes (" " ' ') in JS — straight ASCII only (" ')
- If node --check fails, fix immediately before any other task

## Stack
- Static HTML/CSS/JS + Three.js
- No build step, no bundler — edits are live immediately
- Main files: index.html, style.css, app.js
